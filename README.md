# Cradle of Embers

A Stars Without Number campaign hub — sector map, planet browser, GM tools, and more.

**Live site:** https://wario567.github.io/cradle-of-embers/?seed=cradle-of-embers

## Hosting

Served via GitHub Pages from the `main` branch root. No build step required — open `index.html` directly.

## Development

Clone the repo and open `index.html` in a browser, or use Claude Code to edit and push changes (Pages auto-updates on every push to `main`).
